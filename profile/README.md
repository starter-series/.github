## Every Starter Includes

- **CI/CD** — GitHub Actions: build, test, lint, deploy
- **Security** — gitleaks, npm audit, license checking, scoped permissions
- **Deploy** — Platform-specific publishing pipelines
- **Dependabot** — Grouped updates with lockfile conflict prevention
- **Governance** — Branch protection, PR templates, issue templates

## Quick Start

```bash
gh repo create my-app --template starter-series/docker-deploy-starter
# Write your app, then push — CI/CD handles the rest.
```
